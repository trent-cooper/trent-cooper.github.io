Tailslide is an open-source feature flag framework that makes code deployment easier and safer with circuit breaking capabilities and automated failure recovery.
Tailslide Architecture
Tailslide consists of Tower, a full-stack app for flag and circuit management, NATS JetStream for fault tolerant real-time messaging, an SDK available in multiple languages, and Aerobat + RedisTimeSeries for circuit breaking capabilities.
A feature flag management system purpose-built for new feature deployment with granular control over rollout and developer whitelists.
Flag Dashboard
Flags can be toggled or updated for instant changes within all Tailslide SDK instances. Configure rollout percentages, whitelist developers for dark launches, and keep track of historical changes.
Automated failure protection implemented with a circuit breaker stability pattern gives developers peace of mind with fully automated recovery.
Circuit Dashboard
Get live updates on the health of any enabled circuit. Websockets and a Redis TimeSeries cache give developers a live look at error rates, request counts and the current state of a circuit as it recovers.
